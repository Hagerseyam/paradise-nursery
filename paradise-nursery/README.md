A beautiful React + Redux shopping cart application for indoor houseplants.

## About

Paradise Nursery is your one-stop destination for premium indoor houseplants. We offer a carefully curated selection of beautiful, healthy plants to transform your living spaces into green sanctuaries.

## Features

- **Landing Page**: Welcome page with company branding and easy navigation
- **Product Catalog**: Browse 6 categories with 6+ unique plants each
- **Shopping Cart**: Full-featured cart with quantity management
- **Redux State Management**: Persistent cart state across navigation
- **Responsive Design**: Clean, modern interface

## Tech Stack

- React
- Redux Toolkit
- React Router
- CSS

## Getting Started

1. Install dependencies:
   ```bash
   npm install
   ```

2. Start the development server:
   ```bash
   npm start
   ```

3. Open [http://localhost:3000](http://localhost:3000) in your browser.
